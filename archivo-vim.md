> Este es un archivo creado en vim

Es un editor basado en modos

# Modos

- Modo insert => Insertar texto

- Modo command => Navegar dentro del texto y hacer varias cosas de manera mas rápida

- Modo normal => Navegar por el texto del archivo de manera mas comoda

- Visual => Permite seleccionar texto

Normal: Para entrar en el modo normal se usa esc (Escape)

Comando: Para entrar en modo comando debe de estar en modo normal y despues se ingresa con :

Insertar: Para poder entrar en el modo de insert se usa la tecla i (Insert) y debe de estar en el modo normal

Visual: Se puede acceder con la letra v (Visual)

# Modo comando

En modo comando podemos escribir ciertas letras por ejemplo si estamos en modo comando y escribimos q de (quit) nos permitirá salir fácilmente del editor

q (quit): Salir del editor
w (write): Guardar el contenido si es que este cambio
!: Se pone después de cada letra para forzar la ejecución del comando
q!: Fuerza la salida sin guardar (Salir sin guardar)
w!: Forzar la escritura o el guardado
x: Guardar y cerrar, también se puede con :wq

También se pueden convinar los comandos como wq! (Forzar escritura y salida)

Así mismo vim permite crear tabs, buffers, splits horizontales y verticales, remplazar palabras coincidentes, etc.

Es quizás el editor mas completo y rápido
